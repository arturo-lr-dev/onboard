## Phase 1: MVP (April-May 2026)

**Goal:** Onboard 5 pilot companies

### Features

**Core Platform:**
- Agent profile templates (role, permissions, context)
- Knowledge base builder (upload docs, define processes)
- Access dashboard (view all agents, permissions)
- Basic audit logs

**Onboarding Flow:**
1. Company signs up
2. Define agent role (support, sales, dev, etc.)
3. Upload organizational context
4. Generate agent config (API keys, policies)
5. Agent goes live with full context

**Tech Stack:**
- Frontend: Next.js + Tailwind
- Backend: Node.js + PostgreSQL
- Agent SDK: NPM package (plug-and-play integration)
- Auth: Clerk or Auth0
- Hosting: Vercel + Railway

### Success Metrics
- 5 pilot companies onboarded
- Average onboarding time < 2 hours (vs weeks manually)
- 3/5 pilots convert to paying Beta customers
